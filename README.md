# CUSTOMER_SHOPPING_BEHAVIOR

Customer Shopping Behavior Analysis project focuses on understanding customer purchasing patterns using transactional data

Objective is to extract actionable insights to support data-driven business decisions

Analyzes customer preferences, spending behavior, and sales trends

Demonstrates end-to-end data analytics workflow from raw data to visualization

Tools used include Python, SQL, and Power BI

Python used for data cleaning, preprocessing, and exploratory data analysis

Libraries used in Python include Pandas, NumPy, Matplotlib, and Seaborn

SQL used for data extraction, filtering, aggregation, joins, and transformation

Power BI used to create interactive dashboards and business reports

Dataset includes customer details such as customer ID, age, gender, and location

Transaction data includes order ID, purchase date, quantity, and total amount

Product data includes product name, category, and pricing information

Imported raw data from CSV or database sources

Cleaned missing values and corrected data types using Python

Removed duplicates and handled outliers to improve data quality

Created derived metrics such as total sales, average order value, and purchase frequency

Used SQL queries to create summarized and analytical tables

Performed customer segmentation based on spending behavior and frequency

Analyzed product-wise and category-wise sales performance

Identified top-selling and low-performing products

Conducted monthly and yearly sales trend analysis

Studied repeat purchase behavior and customer loyalty

Built Power BI dashboards with key performance indicators (KPIs)

Visualized total sales, total orders, and total customers

Added interactive filters for time period, category, and customer segments

Displayed charts for sales trends, product performance, and customer insights

Identified high-value customers contributing major revenue

Discovered peak shopping periods and seasonal demand patterns

Highlighted products with consistent and high demand

Provided insights useful for marketing strategy and inventory planning

Project demonstrates practical skills in data analysis, SQL querying, and data visualization

Suitable for Data Analyst and Business Analyst portfolio and resume projects.
